# AIM

# Usage

Simply add the following to your axolotl YAML config:

```yaml
plugins:
  - axolotl.integrations.aim.AimPlugin
```

# Plugin settings

```yaml
aim:
    aim_enable: True
    aim_repo: /aim-repo                # or leave empty to use AIM_REPO env var
    aim_experiment: "my-run"
```
